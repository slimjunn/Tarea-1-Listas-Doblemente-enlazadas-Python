## Descripción

Proyecto que implementa una Lista Doblemente Enlazada en Python con:

-   Inserción al principio
-   Inserción al final
-   Eliminación por carnet
-   Visualización en consola
-   Representación gráfica usando Graphviz

------------------------------------------------------------------------

## Requisitos

-   Python 3.x
-   Graphviz instalado en el sistema
-   Librería graphviz para Python

Instalar librería:

pip install graphviz

Descargar Graphviz:

https://graphviz.org/download/

------------------------------------------------------------------------

## Cómo ejecutar

1.  Abrir terminal en la carpeta del proyecto.
2.  Ejecutar:

python main.py

Las imágenes generadas se guardan en la carpeta:

imagenes/
